# akn.me.uk

**URL:** [https://akn.me.uk](https://akn.me.uk)
**Type:** Personal portfolio
**Stack:** Next.js, Chakra UI

---

## Overview

`akn.me.uk` is the personal portfolio and online home of **Ahnaf Kabir**, a designer and developer based in London, UK. The site acts as a central identity page for the akn ecosystem — linking to projects, work, and commissions.

The tagline is **"Building things that click."** — a statement of design philosophy that emphasises purposeful, intuitive, and refined user experiences.

---

## Design & Technical Details

### Visual Design

- Animated gradient blob backgrounds in yellow, green, and red
- Dark/light mode toggle with theme persistence via `localStorage`
- Smooth fade-in animations with staggered timing for text elements
- Responsive layout optimised for mobile, tablet, and desktop viewports

### Technology

| Layer | Technology |
|-------|-----------|
| Framework | Next.js (React) |
| Component library | Chakra UI |
| Styling | CSS animations & transitions |
| Theme | Persisted via localStorage |

### Navigation

- Single-page layout with scrollable sections
- **Command palette** accessible via the command icon button in the left sidebar — allows keyboard-driven navigation across the site

## Access

The site is public and requires no authentication. Commission enquiries can be made via contact information linked from the site.
